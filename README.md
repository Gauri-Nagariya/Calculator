## 📌 Overview

A lightweight, fully functional calculator web app built with pure **HTML, CSS, and JavaScript** — no libraries, no frameworks, no build tools. Supports all essential arithmetic operations with a clean, user-friendly interface.

---

## ✨ Features

- ➕ **Addition**
- ➖ **Subtraction**
- ✖️ **Multiplication**
- ➗ **Division**
- 🔢 **Decimal support**
- ⌫ **Clear / Reset display**
- 📱 **Responsive Design** — works on all screen sizes
- ⚡ **Zero Dependencies** — open `calc.html` and it just works

---

## 🛠️ Tech Stack

| Tech | Purpose |
|---|---|
| HTML5 | Calculator structure & layout |
| CSS3 | Styling, button grid & display |
| JavaScript (Vanilla) | Arithmetic logic & DOM manipulation |

## 🚀 Getting Started

No installation or setup needed.

### Option 1 — Open directly in browser

```bash
# Clone the repository
git clone https://github.com/Gauri-Nagariya/Calculator.git

# Navigate into the project
cd Calculator

# Open in browser
open calc.html
```
